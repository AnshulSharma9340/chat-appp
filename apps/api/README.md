# ChatApp Application

**Started On:** 01/05/2026

## Tech Stack

### Backend
- Spring Boot
- Java
- MongoDB Database


## Project Description
ChatApp is a real-time chat application built using Spring Boot for the backend and React with TypeScript for the frontend. MongoDB is used as the database to store chat rooms, messages, and user data.

## Features
- Real-time messaging
- Chat room creation
- User-friendly interface
- Responsive design
- Backend REST APIs
- MongoDB database integration

## Installation

### Backend Setup
1. Clone the repository
2. Open the backend project in VS Code / IntelliJ
3. Configure MongoDB connection in `application.properties`
4. Run the Spring Boot application

npm install
